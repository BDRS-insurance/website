# BDRS Associates Insurance Agency 🛡️

A premium, high-performance web application for **BDRS Associates**, a licensed insurance agency dedicated to building dependable risk solutions. This site provides a seamless interface for clients to explore protection plans and request professional consultations.

---

## ✨ Key Features

* **Premium Dark UI:** A sophisticated "Midnight" aesthetic featuring glassmorphism, depth-based shadows, and a clean, professional color palette.
* **Custom Protection Plan Dropdown:** Replaced standard browser-default menus with a bespoke, animated React component for a high-end mobile and desktop "feel."
* **Integrated Inquiry System:** Powered by Web3Forms to deliver client requests directly to the BDRS team without needing a complex backend.
* **Accredited Partners Gallery:** An interactive showcase of major Philippine insurance providers including Pacific Cross, Paramount, and PhilBritish.
* **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop viewing.

---

## 🛠️ Tech Stack

* **Framework:** [React.js](https://reactjs.org/) (Vite)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Animations:** Tailwind Animate & Framer-style transitions
* **Forms:** [Web3Forms API](https://web3forms.com/)
